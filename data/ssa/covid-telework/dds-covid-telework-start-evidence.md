# DDS COVID Telework Start Evidence

## Bottom line

Public sources support a DDS-specific work-from-home start window of March
17-27, 2020. Massachusetts/MRC executive-branch telework began in the same
mid-March emergency period, but no public Massachusetts DDS-specific directive
with an exact start date was located. The remaining MassAbility PRR should
request the exact directive and remote-access records.

## Evidence chain

| Date / window | Source | What it supports |
|---|---|---|
| 2020-03-17 | SSA press release, 2020-03-16 | SSA closed local offices to the public beginning March 17, 2020 and shifted service to remote, phone, and online channels. |
| March 2020; within ten days | SSA congressional testimony, 2021-04-29 | SSA directed employees to work from home in March 2020; only two DDS sites could telework immediately; SSA established a DDS VPN option within ten days so state DDS employees could work from home using their desktop computers. |
| March 2020 | SSA OIG report A-01-20-50963 | SSA allowed DDS employees to move SSA-provided equipment to remote locations during the pandemic emergency. |
| By 2020-09-10 | MRC State Rehabilitation Council minutes | MRC was tracking COVID-19 telework assignments and a Remote Access Survey; this supports a MassAbility/MRC telework process but does not establish the original start date. |

## Working inference

- MassAbility/MRC broad telework start: mid-March 2020, likely March 16-18
  under executive-branch pandemic guidance.
- DDS-specific remote-work start: March 17-27, 2020, based on SSA's local
  office closure and the DDS VPN deployment within ten days.
- Exact Massachusetts DDS start date remains a records-request target.

## PRR target

Request records sufficient to show the exact date MassAbility/MRC DDS employees
were first directed, authorized, or required to work from home/telework due to
COVID-19, including March 2020 emails, memoranda, directives, remote-access/VPN
rollout notices, equipment pickup instructions, Remote Access Survey materials,
and return-to-office plans.
