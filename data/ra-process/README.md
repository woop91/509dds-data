# data/ra-process

Reference materials for the **reasonable-accommodation (RA) process** as it applies
to Massachusetts state employees, with focus on MassAbility DDS members covered by
SEIU Local 509: the official MOD ADA-coordinator listing, a compiled contact
directory, mass.gov captures of the RA request/appeal process, disability-rights
legal references, a Disability Law Center self-advocacy guide, and employer context
for MassAbility DDS.

## Machine-consumption conventions

Each file is paired with a `<file>.meta.json` data card conforming to
[`../../schemas/dataset.meta.schema.json`](../../schemas/dataset.meta.schema.json).
The repo-wide index is [`../../catalog.json`](../../catalog.json) /
[`../../llms.txt`](../../llms.txt).
